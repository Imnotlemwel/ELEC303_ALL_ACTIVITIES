## Activity Descriptions

### Activity 1: Palmer Penguins Dataset  
Explore the Palmer Penguins dataset to practice fundamental data cleaning, visualization, and statistical techniques. Focus on analyzing species characteristics and relationships.

---

### Activity 2: UK Road Accident Data  
Analyze UK road accident data using exploratory data analysis to identify trends, patterns, and contributing factors to traffic incidents.

---

### Midterm Exam: Chicago Crime Analysis  
Conduct spatial and temporal analysis of Chicago crime data. Techniques include aggregation by crime type, visualization on maps, and temporal trend analysis.

---

### Project 3: Dengue Analysis with Linear Regression  
Perform time-series and regional analysis of dengue cases in the Philippines using linear regression and data visualization to identify disease trends and hotspots.

---

### Assignment: Linear Regression  
Implement logistic regression for binary classification tasks, covering model building, evaluation, and prediction.

---

### Titanic Survivability Prediction  
Develop machine learning models to predict Titanic passenger survival, including feature engineering and model evaluation.

---

### Machine Learning Exercise 1: Housing Price Prediction  
Practice core machine learning workflows by predicting housing prices using regression models, focusing on training, evaluation, and prediction.
